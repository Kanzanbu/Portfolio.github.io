Introduction

Welcome to my projects page! Here, you'll find a collection of web development projects I've worked on. Each project showcases different skills and technologies I've learned. From simple websites to interactive applications, these represent my growth as a developer.

Projects

Calculator App

A simple calculator application built with HTML, CSS, and JavaScript. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

This project demonstrates fundamental JavaScript event handling and DOM manipulation. It's responsive and works on both desktop and mobile devices.

Jot Haven

A note-taking web app called Jot Haven, designed for jotting down ideas quickly. Features include creating, editing, and deleting notes, with local storage for persistence.

Built with React for dynamic UI updates and CSS for styling. This app taught me about state management and user interface design.

Spotify App

A music player app inspired by Spotify, featuring playlist creation, song search, and playback controls. It integrates with a music API for song data.

Developed using JavaScript, HTML, and CSS with audio API integration. This project enhanced my skills in API consumption and media handling.
